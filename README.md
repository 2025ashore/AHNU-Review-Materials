AHNU-Review-Materials
安徽师范大学（AHNU）复习资料汇总库，旨在为安徽师范大学的同学们提供各专业、各课程的复习参考资料，助力大家高效备考。
📚 项目简介
本仓库收集整理了安徽师范大学不同学院、不同课程的复习资料，包括但不限于：
课程笔记（课堂重点、知识点梳理）
习题集与参考答案（课后题、往年考题、模拟题）
考试重点总结（老师划重点、高频考点归纳）
课件与补充资料（部分课程的课件整理、拓展阅读材料）
资料主要由在校同学自发整理与贡献，力求覆盖公共基础课、专业基础课及部分专业课，供大家学习参考。
📖 使用说明
获取资料：
直接在线浏览：进入对应课程目录，点击文件即可查看。
本地下载：克隆仓库到本地，方便离线查看：
bash
运行
git clone https://github.com/2025ashore/AHNU-Review-Materials.git
查找资料：
按目录结构逐层查找对应学院 / 课程。
利用 GitHub 的「Find file」功能（页面上方搜索框），直接搜索课程名称关键词。
🤝 贡献指南
欢迎各位 AHNU 同学补充和完善资料，贡献方式如下：
Fork 本仓库到自己的 GitHub 账号。
上传资料到对应目录（建议按「课程名称 / 资料类型」命名，如「2023 期末真题.pdf」）。
提交 Pull Request（PR），注明资料对应的课程、年级、整理人（可选）。
贡献要求：
资料需为原创整理或已获得授权的内容，禁止上传侵权材料。
资料需保证一定清晰度和实用性（如笔记条理清晰、考题答案准确）。
避免重复上传相同或相似资料（可先查看仓库现有内容）。
📝 免责声明
本仓库资料仅供学习参考，不代表学校官方观点或考试范围，最终请以课堂教学内容和老师要求为准。
资料版权归原整理者所有，请勿用于商业用途，转载请注明来源。
如资料存在侵权问题，请联系仓库维护者删除。
